Biography

IKFA Cloud 是一个内部脚手架项目，旨在为实践和开发微服务架构的应用提供开发、测试、部署全流程如何在泰康内部 泰康云PASS 平台上提供实践经验；

Interests

  • development：spring stack、DataBase
  • docker、kubernetes、openshift、service mesh
  • monitor

Demos

Welcome to the demos of ikfa cloud. the demos available include:

  • H+ 后台管理系统 (前端基于Html、bootstrap、jquery的后台管理系统)

  • Vue 后台管理系统 (前端基于 Vue 的后台管理系统)

  • IKFA2.0 后台管理系统 (基于IKFA2.0的后台管理系统)

  • 后台管理系统对外服务 (基于spring boot的后台rest接口)

其中，H+ 后台管理系统 与 Vue 后台管理系统 对应的后台管理服务 api 都是基于后台管理系统对外服务.
